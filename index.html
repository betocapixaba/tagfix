Grade de guias

Cria guias uniformemente espaçadas — ótimo para alinhar linhas de uma tabela de preços de uma vez só. Textos "grudam" (snap) nas guias ao arrastar.

Projeto

Salva tudo — imagem, textos pendentes e guias — num arquivo pra continuar de onde parou depois.

Dicas

Réguas e guias: arraste da régua do topo para criar uma guia horizontal, ou da régua da esquerda para uma vertical. Clique numa guia e arraste para movê-la; solte fora da imagem para remover. As guias nunca aparecem na imagem exportada.

Fluxo recomendado: ajuste o texto, clique em Aceitar alteração para gravá-lo na imagem, e siga para o próximo preço.
